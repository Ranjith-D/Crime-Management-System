# Crime-Management-System
A Crime Management System Website.

There are two modules. The admin and a normal user.

The admin can manage the entire website by adding/deleting the crime report/complaints.

The normal user can register and add any complaint with an image and description.

## Prerequisties
* Windows OS
* Java JDK and JRE 
* NetBeans IDE
* Wamp Server for MySQL
* Google chrome Browser
